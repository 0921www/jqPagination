jqPagination

a jQuery pagination plugin (obviously)

jqPagination is a jQuery plugin that gives a slightly different spin on pagination. This pagination method revolves around a main input that allows the user to enter the page they want to be warped to, thus avoiding the usual long list of page numbers. The plugin handles first, previous, next and last links, these can be removed if desired.

    What made you think of this method?

    Strangely the method we currently use for our ERP application mixes an input element for the desired page and a span for the ' of 4 pages' bit... and this looks really bad when the webpage was printed and was difficult to style in general, considering the text was shared between two elements.

    So you came up with the concept on your own?

    Definitely not, I was inspired by something Chris Rhee had Dribbbled, I liked the idea of pagination that had no huge list of numbers, made it very clear to the user what page they were on but also allowed them to enter a very specific and accurate page to jump to. The design I put together was also inspired by the colours in this Dribbble by Shaun Moynihan
